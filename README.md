# Weather MCP Project

This project provides a weather server using Node.js. It is designed to be simple and easy to set up, allowing users to quickly get started.

## Features

- Provides weather data via an MCP server.
- Easy to set up and run locally.
- Extensible for additional features.

## Prerequisites

Before you begin, ensure you have the following installed on your system:

- [Node.js](https://nodejs.org/) (version 14 or higher)
- [npm](https://www.npmjs.com/) (comes with Node.js)

## Getting Started

Follow these steps to set up and run the project:

### 1. Clone the Repository

```bash
git clone https://github.com/victorzxj/weather-mcp.git
cd weather-mcp
```

### 2. Install Dependencies

Run the following command to install the required dependencies:

```bash
npm install
```

### 3. Start the Server

Start the weather server with:

```bash
npm start
```

### 4. Access the Server

The server will start on `http://localhost:3000` by default. You can access it via your browser or API client.

## Project Structure

- `weather-server.js`: The main server file.
